# @storacha/ui-react Examples

This directory contains example implementations demonstrating how to use the components from `@storacha/ui-react`.

## Space Management Components Demo

See `space-management-demo.tsx` for a complete example showing:
- SpaceEnsurer - Ensuring spaces exist
- SpaceCreator - Creating new spaces
- SpaceFinder - Searching and selecting spaces
- SpacesList - Displaying spaces with actions
- SpacesTabNavigation - Tab-based navigation

### Running the Example

This is a reference implementation. To use it:

1. Copy the code into your React application
2. Install `@storacha/ui-react`
3. Provide your connection and service principal
4. Add your own styling (components are unstyled by default)

### Key Patterns Demonstrated

- How components work together
- Proper prop usage
- State management between components
- Filtering spaces by type (public/private)
- Custom callbacks and handlers

## Note

These examples are for reference only and are not part of the compiled package. They demonstrate best practices and common usage patterns.
